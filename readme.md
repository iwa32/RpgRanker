## RpgRanker
好きなキャラクターを選択し、モンスターを倒すゲームです。弱点を突き、素早く倒すことがランキング上位のコツです。

## 主な機能
- キャラクターの一覧を取得する
- 戦うキャラクターを取得する
- 対戦するモンスターを取得する
- 対戦データを保存する
- ランキングページに対戦データを表示させる
- ランキングでのページネーション機能
- 認証機能(会員登録、ログイン、ログアウト)

## 使用技術

### フロントエンド
- Vue.js(Vuex, VueRouter)でのSPA

### バックエンド

#### 使用言語
- Laravel6

#### データベース
- MySQL5.7

#### 開発環境
- Docker(laradockの使用)

#### 本番環境
- heroku
